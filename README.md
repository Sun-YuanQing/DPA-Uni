# DPA-Uni
这个是 深圳傲鹏软件 PDA的项目，已限制不能提交新的代码。
PDA   
---

&emsp;**傲鹏MES制造执行系统 PDA端**（Manufacturing Execution System）   
&emsp;系统主要功能包括: 制造数据管理、计划排程管理、生产调度管理、库存管理、质量管理、人力资源管理、工作中心/设备管理、工具工装管理、采购管理、成本管理、项目看板管理、生产过程控制、底层数据集成分析、上层数据集成分解等管理模块
